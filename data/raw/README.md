# Raw Data Description
**Jingtao Yang**

The raw data from the [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) is used in this project. The dataset contains 9 tables in CSV format, they are:
- `olist_customers_dataset.csv`: Information about customers
- `olist_geolocation_dataset.csv`: Geographical information
- `olist_order_items_dataset.csv`: Details of items in each order
- `olist_order_payments_dataset.csv`: Payment information for orders
- `olist_order_reviews_dataset.csv`: Customer reviews for orders
- `olist_orders_dataset.csv`: Main order information
- `olist_products_dataset.csv`: Information about products
- `olist_sellers_dataset.csv`: Information about sellers
- `product_category_name_translation.csv`: Translation from Portuguese to English

The raw data is stored in the `data/olist_ecommerce` directory locally, which will not be uploaded to Github.
